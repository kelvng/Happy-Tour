Welcome to Happy Tour

Khi làn sóng đại dịch COVID-19 kết thúc, nhu cầu du lịch sẽ bùng nổ trên toàn thế giới trong đó có Việt Nam. Để đón đầu nhu cầu du lịch ngày càng cao và đảm bảo chất lượng dịch vụ, quản lý hiệu quả và tiếp cận khách hàng dễ dàng hơn, chúng tôi đã triển khai hệ thống quản lý và đặt tour du lịch - Happy Tour . Giúp khách hàng dễ dàng đặt vé trực tuyến và giúp chủ doanh nghiệp dễ dàng quản lý đội ngũ nhân viên và các tour du lịch. Từng bước tin học hóa mọi thông tin dữ liệu và quản lý và thay thế dần các công tác quản lý thủ công của cửa hàng sang trực tuyến. Qua đó đem lại nguồn doanh thu lớn hơn cho doanh nghiệp

🚦 How to install

  1. Cài đặt NodeJS

Install NodeJS

  1. Cài đặt MongoDB

Ở file config.env thay đổi thông tin database document

DATABASE=mongodb+srv://truongwfngo:<PASSWORD>@cluster0.lym5b.mongodb.net/HappyTour?
JWT_SECRET=Ub3nappGV08yCdGVcB/Zrb8eTZUNBRyzYPWUvkprvbg

Ở file config.env thay đổi thông tin email

EMAIL_USERNAME=domain@gmail.com
EMAIL_PASSWORD=<password>
  1. Cài đặt các gói npm hỗ trợ trong file package.json (dependencies)
npm install
  1. Chạy project ở môi trường dev: 
npm start
  1. Chạy project ở môi trường product: 
npm run start:prod
